# AGENT.md

## О проекте

`sarmaexpress` - монорепозиторий сайта и сервисных flow компании:

1. `Сарма Экспресс` - публичная витрина, калькулятор, страницы услуг и карта пунктов выдачи.
2. `SUPERBOX` - прикладной интерфейс для доставки из интернет-магазинов, заказа по ссылке, отслеживания, отмены заказов, тарифов и бизнес-сценариев.

Пользователь работает на русском. Handoff, статусы и ответы по проекту лучше вести на русском.

## Маркетплейс-анкеты

Данные брались из `Формы_фух.xlsx`.

В `pickup_paid` настроены анкеты и информационные блоки для:

- Wildberries
- Ozon
- Яндекс Маркет
- Lamoda
- Авито
- 5Post
- DPD
- Детский Мир
- Золотое Яблоко
- Лэтуаль
- СДЭК

Для этих сценариев используется выбор только 8 клиентских пунктов выдачи из таблицы. После выбора пункта показывается адрес/ПВЗ/терминал в Ростове-на-Дону, куда пользователь должен оформить заказ. Для WB и Ozon адрес зависит от выбранного пункта, для остальных чаще фиксированный.

Кнопка отправки в маркетплейс-анкетах активна только после согласия с условиями.

## Заказ по ссылке

`pickup_standard` работает как `Заказ по ссылке` и доступен для:

- Wildberries
- Ozon
- Яндекс Маркет

Поля: имя, фамилия, телефон, размер, ссылка на товар, пункт выдачи, дополнительная информация. Есть условия: комиссия за выкуп 15%, возврат бесплатно. Кнопка `Сделать заказ` активна только после согласия.

В этом сценарии также есть выбор 8 пунктов выдачи и блок с адресом Ростовского ПВЗ/точки, куда заказывать.

## Отмена заказа

В меню `Услуги` добавлен пункт `Отмена заказа` со страницей `/cancel-order`.

Логика использует существующий backend cancel API:

1. ввод номера заказа;
2. предупреждение про поддержку, если заказ оформлен на нашего получателя;
3. проверка номера;
4. подтверждение `Да / Нет`;
5. после `Да` заказ переводится в статус `CANCELLED`.
