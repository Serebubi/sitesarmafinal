# CURRENT_STATUS.md

## Статус проекта на 2026-04-23

Текущая ветка: `codex/ver7`.

Проект находится на этапе выравнивания публичной оболочки `Сарма Экспресс` и ключевых `SUPERBOX flow` под один визуальный стиль.

## Что завершено

### Главная и навигация

- `/` рендерит `SarmaExpressPage`.
- Верхняя плашка вынесена в `frontend/components/sarma-express-header.tsx`.
- Навигация актуальна:
  - `Калькулятор` -> `/calculator`
  - `Отслеживание` -> `/superbox?flow=order_lookup`
  - `Бизнесу` -> `/superbox?flow=business`
  - `Тарифы` -> `/superbox?flow=tariffs`
  - `Доставка из интернет-магазинов` -> `/superbox?flow=pickup_paid`
  - `Отправления в РФ` -> `/superbox?flow=ship_russia`
  - `Пункты выдачи` -> `/pickup-points`
- На главной все 4 карточки `Наши услуги` ведут на `/superbox?flow=pickup_paid`.

### SUPERBOX в Sarma-стиле

В `frontend/components/superbox-app.tsx` уже переведены на новый стиль:

- `pickup_paid` шаг 1: выбор маркетплейсов и служб доставки.
- `pickup_paid` шаг 2: анкета после выбора маркетплейса.
- `pickup_standard` шаг 1: выбор маркетплейса для заказа по ссылке.
- `pickup_standard` шаг 2: анкета заказа по ссылке.
- `order_lookup`: отслеживание посылки.
- `ship_russia`: отправления в РФ.
- `tariffs`: тарифы.
- `business`: пустая страница с текстом `скоро добавим)`.

Исправлено мигание старого интерфейса: `activeFlow` инициализируется сразу из `?flow=...`, а не через поздний `useEffect`.
